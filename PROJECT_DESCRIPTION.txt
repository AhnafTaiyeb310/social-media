## 3. GITHUB README VERSION (Comprehensive & Technical)
# Aura Social Media Platform

Aura is a full-stack social media application built with a modern tech stack, focusing on architectural integrity, performance, and a polished user interface.

### 🚀 Features

#### Content Management
*   **Full CRUD**: Create, read, update, and delete posts with ease.
*   **Drafts System**: Save your progress and keep ideas private until they are ready for the world.
*   **Post Scheduling**: Schedule your content to go live at a specific date and time.
*   **Advanced Media**: Support for multiple image uploads per post with asynchronous processing.

#### User Engagement
*   **Nested Discussions**: Real-time comment system with support for infinite nested replies.
*   **Engagement Metrics**: Like and comment counts updated instantly across the UI.
*   **Follow System**: Follow your favorite authors and get tailored suggestions.

#### Discovery
*   **Unified Global Search**: Search for posts, people, and tags in one place.
*   **Tagging System**: Categorize content with custom tags for better discoverability.
*   **Trending Content**: Dynamic "Trending Articles" and "Popular Topics" sidebars.

### 🛠 Tech Stack

**Frontend:**
*   **Framework**: Next.js 14+ (App Router)
*   **State Management**: Zustand (with Persistence)
*   **Data Fetching**: TanStack Query (React Query)
*   **UI Library**: Tailwind CSS & Preline UI
*   **Icons**: Lucide & React Icons

**Backend:**
*   **Framework**: Django 5.0 & Django REST Framework
*   **Authentication**: SimpleJWT (JSON Web Tokens)
*   **Database**: PostgreSQL / SQLite
*   **Task Queue**: Celery & Redis (for async image uploads)
*   **Storage**: Cloudinary (Image Hosting)

### 🏗 Architecture
The project follows a decoupled architecture where the Django backend serves as a stateless API and the Next.js frontend handles the rendering and client-side logic. 
*   **Optimized Queries**: Uses `select_related` and `prefetch_related` to solve N+1 problems.
*   **Background Processing**: Offloads heavy image processing to Celery workers to keep the UI responsive.
*   **Persistent Auth**: Implements a secure token refresh interceptor in Axios to manage sessions automatically.
